## For the Go Community

### Weiqi Roadmap

A structured guide and resource library for learning Go/Baduk/Weiqi, especially useful for beginners and double-digit kyu players. [Explore the roadmap](https://weiqi.soumyak4.in).
 - Changelog [Here](/project/Weiqi-Roadmap/)

### BadukTube

A searchable collection of free Go lectures and study series. [Browse BadukTube](https://baduktube.soumyak4.in).
 - Changelog [Here](/project/BadukTube/)

### TsumeGod

A Telegram tsumego bot for quick, daily Go-puzzle practice. [Start practising](https://tsumegod.soumyak4.in).
 - Changelog [Here](/project/TsumeGod/)
 - Inspired by _dontdiehard/Skwidder_(Discord) : <a href="https://github.com/Skwidder/IGS-Puzzle-Bot" target="_blank">GitHub</a>

### OGS Script

Enhances the Online-Go.com board and review experience.
- [Get the Tampermonkey script](https://github.com/SoumyaK4/OGS-Tampermonkey)

### Sensei's Library Script

A Tampermonkey script that gives Sensei's Library a clean dark theme and opens SGFs in a compact BesoGo popup.
- [Get the script on GitHub Gist](https://gist.github.com/SoumyaK4/527634f02694d3de9e73123f9c611726)

### SWHub

An offline-first Go training app with pattern search, spaced repetition, and P2P tsumego battles.
- [Download SWHub / WeiqiHub](https://github.com/SoumyaK4/WeiqiHub/releases/)
- [Read the project overview](/blog/weiqihub/)

### Weiqi Tracker

Track your game data and make your progress visible over time.
- [Open Weiqi Tracker](https://wq-tracker.soumyak4.in/)

### SGF to Music Video

Turn an SGF game record into an elegant music video.
- [Create an SGF music video](https://sgf2mv.soumyak4.in/)

### Web KataTrain

A contribution that brings KataTrain into the browser.
- [View the project on GitHub](https://github.com/SoumyaK4/web-katrain)

### Go Timer

An installable, offline-friendly Go clock that supports major time-control systems.
- [Open Go Timer](https://gotimer.soumyak4.in/)

### Katrain Theme

BadukTV theme with JeffSounds
- [Katrain Theme](https://github.com/SoumyaK4/Katrain-Theme-SoumyaK4)

### Free KataGo Guide

A practical guide to running KataGo on free cloud GPUs.
- [Read the guide](https://gist.github.com/SoumyaK4/87336ad7553f4a80173ee477ebeba364)
